### Added
- Initial release of RoleMirror framework.
- Dual-Agent simulation architecture (Mock NPC & Player Simulator).
- LLM-as-a-Judge grading system with configurable criteria.
- Pass@K and Pass^K evaluation metrics.
- CLI tools for online evaluation (`main.py`) and offline log analysis (`evaluate_logs.py`).
- Visualization tools for reporting (`analyze_report.py`).
- Modular configuration for NPCs (`npcs.json`) and Scenarios (`scenarios.json`).
